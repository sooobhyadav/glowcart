# GlowCart

GlowCart is a simple React + Vite based cosmetic product showcase website.

## Features

- Responsive product grid layout
- Reusable React components
- Props-based product cards
- Custom button component
- Modern UI design
- Product data using custom JSON array

## Tech Stack

- React JS
- Vite
- CSS3
- JavaScript (ES6)
